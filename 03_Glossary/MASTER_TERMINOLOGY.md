# MASTER_TERMINOLOGY - Glossario mestre de termos recorrentes

## Controle

- Versao: 0.1
- Data: 2026-06-26
- Autor: JP-BR Insurance Knowledge Base
- Historico: Criacao inicial.

## Finalidade

Este documento funciona como autoridade terminologica para termos que aparecem em mais de uma categoria de seguro.

Ele nao substitui os verbetes por produto. Ele define a traducao principal e orienta quando cada categoria pode ter explicacao propria.

## Regra geral

Quando um termo japones aparecer em mais de uma categoria, manter:

- uma traducao principal
- notas de uso por produto
- referencias cruzadas para os verbetes existentes

Nunca criar traducao alternativa sem justificativa editorial.

## Alertas de traducao

- `保険料` deve ser explicado como valor pago pelo seguro. Evitar confundir com "indenizacao".
- `申込書` nao deve ser explicado como contrato final. E proposta/pedido.
- `告知義務` nao deve ser suavizado demais. O cliente precisa entender risco de informacao incorreta.
- `不支払通知` deve ser tratado com cuidado comercial: explicar motivo, verificar documentos e avaliar reanalise sem prometer reversao.
