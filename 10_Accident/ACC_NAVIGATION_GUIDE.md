# ACC_NAVIGATION_GUIDE - 傷害保険・事故対応ナビゲーション

## 管理情報

- Versão: 0.1
- Data: 2026-06-29
- Autor: JP-BR Insurance Knowledge Base
- Histórico: Criação inicial do guia de navegação da categoria ACC.

## 目的

傷害保険、労災保険、通勤災害、事故通知、診断書、診療明細書、入院保険金、手術保険金、免責日数、免責事由を、ブラジル人顧客へ説明するための検索導線。日本語で対応方針を確認し、必要に応じてポルトガル語の説明文とカタカナ読みを使って顧客に説明する。

## 推奨確認順序

1. `ACC-0001` で民間の傷害保険を確認する。
2. `ACC-0002` と `ACC-0003` で労災保険、通勤災害、民間保険を分ける。
3. `ACC-0004` で事故通知の流れを確認する。
4. `ACC-0005` と `ACC-0006` で診断書と診療明細書を確認する。
5. `ACC-0007` と `ACC-0008` で入院・手術の給付を確認する。
6. `ACC-0009` と `ACC-0010` で免責日数と免責事由を確認する。
7. `ACC-0011` から `ACC-0013` で通院、死亡、後遺障害の給付を確認する。
8. `ACC-0014` から `ACC-0016` で請求書類、事故証明書、領収書を確認する。
9. `ACC-0017` から `ACC-0019` で急激・偶然・外来の事故要件を確認する。
10. `ACC-0020` で事故証明を概念として整理し、`事故証明書` と分ける。

## NotebookLMでの検索例

- ブラジル人のお客様に傷害保険をどう説明しますか。
- 仕事中の事故は労災保険と民間保険をどう分けますか。
- 通勤中の事故をポルトガル語で説明したいです。
- 診断書と診療明細書の違いを説明したいです。
- 免責日数をブラジル人顧客にどう説明しますか。
- 飲酒中の事故で支払可否を断定しない説明をしたいです。

## 対応上の注意

- 労災保険、公的制度、健康保険、民間の傷害保険を混同しない。
- 日本語の技術理解と、顧客に伝えるポルトガル語を分ける。
- ポルトガル語の発話は、カタカナ読みを確認してから使う。
- 契約条件、証券、事故原因、勤務先、公的制度、保険会社の確認前に断定しない。
- 迷う場合はFAQ、DIALOGUE、CASEの順で確認する。
